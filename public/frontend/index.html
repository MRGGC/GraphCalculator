simple calculator










+ : add, գումարել

- : subtract, հանել

* : multiply, բազմապատկել

/ : divide, բաժանել

% : percentage (first number % second number, a-ն b-ի քանի տոկոսն է)

/? : dividable or not, a-ն b-ի բաժանվում է, թե ոչ

sqrt : square root, քառակուսի արմատ (write only in the a, գրել միայն a-ի մեջ)

2 : square up, քառակուսի բարձրացնել (You can write just 2 in the a input, կարող եք միայն 2 գրել a թվի համար)

n : n powers up, n աստիճան բարձրացնել (You can write just n, a is powered to b, կարող եք միայն n գրել, a թիվը b աստիճան է բարձրանում)
